### SKILL.md 新增内容

#### 触发语
- `python，重新学习 XX` - 将指定章节状态改为"重新学习"（XX 为课程序号或名称）

#### 状态管理规则
- 重新学习时：状态改为"重新学习"，分数保留原分数
- 完成后：状态改回"已完成"，分数取最高分

#### 输出格式
```
## 重新学习

**课程**：XX-课程名
**原分数**：XX分
**状态**：已改为重新学习

完成作业后输入 `python，提交作业` 提交，分数将取最高分。
```

#### 打分逻辑
```python
# 重新学习完成后打分
if 原状态 == "重新学习":
    新分数 = max(原分数, 新分数)
```
