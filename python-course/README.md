# Python 学习课程

## 项目介绍

这是一个 Python 学习课程项目，包含 20 个章节，从基础语法到高级特性，帮助你系统地学习 Python 编程。

**课程来源**：
- 廖雪峰 Python 教程：https://liaoxuefeng.com/books/python/introduction/index.html
- 菜鸟教程 Python3：https://www.runoob.com/python3/python3-tutorial.html

**课程内容**：
- 20 个章节，涵盖 Python 基础到高级特性
- 每个章节包含：课程内容（README.md）、示例代码（demo.py）、作业（homework.py）
- 支持学习进度跟踪和作业评分

## 功能说明

### 上传更新

**触发语**：`上传更新`

**功能**：保存学习进度并推送到 GitHub

**包含内容**：
- 课程进度（COURSE_INDEX.md）
- 作业文件（homework.py）

**执行流程**：
1. 读取当前学习进度
2. 添加进度和作业文件到 git
3. 自动生成 commit message
4. 拉取远程更新（git pull --rebase）
5. 推送到远程（git push）

**commit message 格式**：
- 有分数：`更新进度：XX-课程名 XX分`
- 无分数：`更新进度：XX-课程名 进行中`

**示例**：
```
上传更新
```

### 重新学习

**触发语**：`python，重新学习 XX`（XX 为课程序号，例如：`python，重新学习 01`）

**功能**：将已完成的章节状态改为"重新学习"，保留原分数

**使用场景**：
- 对已完成的章节学习不满意
- 想要重新巩固某个知识点

**执行流程**：
1. 读取当前学习进度
2. 找到指定章节
3. 检查状态是否为"已完成"
4. 将状态改为"重新学习"，保留原分数

**重新学习完成后**：
- 提交作业时，分数取原分数和新分数的最高分
- 状态改回"已完成"

**示例**：
```
python，重新学习 01    # 重新学习第一个Python程序
python，重新学习 05    # 重新学习 list和tuple
```

## 使用方法

### 基本流程

1. **开始学习**：输入 `python，开始工作` 或 `python，下一课` 获取下一节课程
2. **学习内容**：查看 README.md 了解知识点，运行 demo.py 查看示例
3. **完成作业**：编辑 homework.py 完成作业题目
4. **提交作业**：输入 `python，提交作业` 提交并获取评分
5. **保存进度**：输入 `上传更新` 保存学习进度到 GitHub

### 查看进度

输入 `python，进度` 查看当前学习进度和分数。

### 重新学习

对已完成的章节不满意时，输入 `python，重新学习 XX` 重新学习。

## 评分标准

### 运行结果（40分）
- 代码能正常运行，无语法错误：20分
- 输出结果正确：20分

### 代码规范（30分）
- 命名规范（变量、函数名清晰）：10分
- 注释充分（关键逻辑有注释）：10分
- 代码格式（缩进、空行合理）：10分

### 完成度（30分）
- 所有作业题目都完成：20分
- 代码实现方式合理：10分
