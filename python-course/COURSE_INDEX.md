## 课程大纲

### 第一阶段：Python 基础（01-08）
- 01-04：环境搭建、基础语法、数据类型、字符串
- 05-06：列表、元组、字典、集合
- 07-08：条件判断、循环语句

### 第二阶段：函数和高级特性（09-13）
- 09-11：函数定义、参数、递归
- 12-13：切片、迭代、列表生成式、生成器

### 第三阶段：函数式编程和模块（14-16）
- 14-15：高阶函数、装饰器、偏函数
- 16：模块导入和包管理

### 第四阶段：面向对象编程（17-18）
- 17：类和实例、继承和多态
- 18：高级特性、定制类

### 第五阶段：错误处理和IO（19-20）
- 19：异常处理、调试、单元测试
- 20：文件读写、路径处理

## 使用说明

1. 输入 `python，开始工作` 或 `python，下一课` 获取下一节课程
2. 学习课程内容，查看 demo.py 示例代码
3. 完成 homework.py 中的作业
4. 输入 `python，提交作业` 提交并获取评分
